# 算符优先分析法

#### 题目

编程实现以下文法 `G[E]` 的算符优先分析法：

```none
E -> E '+' T | T
T -> T '*' F | F
F -> '(' E ')' | 'i'
```

|      |  +   |  *   |  i   |  (   |  )   |
| :--: | :--: | :--: | :--: | :--: | :--: |
|  +   | >-1  |  <1  |  <   |  <   |  >   |
|  *   |  >   |  >   |  <   |  <   |  >   |
|  i   |  >   |  >   |      | 1000 |  >   |
|  (   |  <   |  <   |  <   |  <   |  =0  |
|  )   |  >   |  >   |      |      |  >   |
